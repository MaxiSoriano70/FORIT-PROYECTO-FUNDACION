Editar Datos Personales
El nombre es obligatorio. Debe tener al menos 2 caracteres. Solo se permiten letras y espacios.
El apellido es obligatorio. Debe tener al menos 2 caracteres. Solo se permiten letras y espacios.
El teléfono es obligatorio. Debe tener al menos 8 dígitos. Debe tener como máximo 15 dígitos. Ingrese un número válido (puede incluir +).
El domicilio es obligatorio. Debe tener al menos 3 caracteres. Solo se permiten letras, números y algunos símbolos (.,#-).
El email es obligatorio. Ingrese un email válido.
Ingrese una URL válida (jpg, jpeg, png, gif, webp).
